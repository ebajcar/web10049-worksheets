List 8: Description list

term
definition of the term
client
computer or a virtual machine that makes requests for service from a server on behalf of the user
server
computer or a vitual machine that processes requests from the server and provide services such as web pages and database data
http
protocol running on server servicing requests for web pages
sftp
secure file transfer protocol to upload and download files between client and server.
html
hypertext markup language used to organize content for presentation on a web page.
